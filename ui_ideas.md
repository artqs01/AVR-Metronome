```
______________________
|tempo:        123bpm|
|time sig:    127/128|
|subdivisions: 256ths|
|beat:            127|
|menu:              x|
+--------------------+
```

```
______________________
|Menu:               |
|sounds & loudness: x|
|polyrhythm:        x|
|polimeter:         x|
```
